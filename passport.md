# Паспорт проекта

## Название
Учёт обращений студентов

## Цель
Создать простой web-проект для добавления обращений студентов в список.

## Структура проекта
- `index.html` — основная страница
- `style.css` — оформление страницы
- `script.js` — простой JavaScript-код
- `passport.md` — описание проекта

## Функции
- ввод текста обращения;
- добавление обращения в список.

## Используемые технологии
- HTML
- CSS
- JavaScript

## Результат
Рабочий web-проект с простой структурой и базовой вёрсткой по теме «Учёт обращений студентов».
